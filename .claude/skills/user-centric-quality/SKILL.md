# User-Centric Quality (Outcome-Based)

Use this skill when tracking quality based on customer outcomes instead of internal vanity metrics.

## Tools
- Product analytics funnels
- Customer support defect tagging
- Satisfaction/experience dashboards

## Script
- `bash .claude/skills/shared/scripts/generate_quality_report.sh user-centric-quality 30`

## Metrics
- Customer-Reported Defects (escaped defects)
- User Error Rate / Failed Journeys
- Apdex / User Satisfaction Score
- Feature Adoption Rate
- Churn due to Product Issues
